8.6. Genetic Codes¶

Fundmental information encoding system and a fascinating subject of study. cogent3 provides a dedicated object for handling genetic code information. The genetic codes included with cogent3 are indicated in the following table indicates.

Genetic code objects act like dictionaries for trinucleotide strings or single letter strings. The former are interpreted as RNA or DNA, the latter as the single code amino acid.

You can get the encoded amino acid from a RNA triplet

aa = gc["UAC"]
aa

'Y'

or DNA triplet.

aa = gc["TAC"]
aa

'Y'

The mapping from codon to amino acid is provided by a sense_codons attribute of genetic code instance. So taking the list of that dict returns just the sense codons 1.

1

I’m truncating that list to just 4 elements to simplify the display.

list(gc.sense_codons)[:4]

['TTT', 'TTC', 'TTA', 'TTG']

You can get all the codons that encode an amino acid.

codons = gc["Y"]
codons

['TAT', 'TAC']

You can check whether a codon is a start

gc.is_start("ATG")

True

or stop codon

gc.is_start("TAA")

False

Stop codons are represented by "*" character.

gc["TGA"]

'*'

gc.translate("TCGACCGTTTAAGCC")

8.6.1. Exercises¶

Identify all sense codons that differ from each other at only one of the codon positions. Group these pairs by codon position. The following questions refer to these groupings.

  1. Pick a genetic code and, for each such codon position group, count the number of changes that are synonymous. Is there a difference between the codon position and the proportion of synonymous changes?

  2. Does the property measured in 2.) vary between the genetic codes?

  3. Now categorise the codon differences by whether they are a transition or transversion change. Assess whether the fraction of synonymous changes differs between transition and transversion changes.

  4. Is there variation (between the genetic codes) in the number of stop codons? Assess this programmatically.

    Hint: look at the attributes on the genetic code instance.
