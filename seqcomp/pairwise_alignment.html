But we typically do not observe the process and must infer an alignment that can be applied to deduce what happened.

seq1  ACAGT

OR

seq1  ACAGT
seq2  --A-T

There are optimally efficient algorithms available for pairwise alignment. We first tackle the problem of global alignment.

6.13.1. Needleman and Wunsch (NW) 1¶

1(1,2)

Needleman & Wunsch (1970). A general method applicable to the search for similarities in the amino acid sequence of two proteins. Journal of Molecular Biology, 48: 443–453

Compared to the dotplot approach, it allows for more sophisticated “match”, i.e. the sequence state may be different but still considered a match. It efficiently (in terms of computation) identifies the optimal “path” through the matrix.

6.13.1.1. The scoring system¶

We will start by using the exact same scoring matrix as NW 1:

  • states matched, score = 1

  • states mismatched, score = -1

  • gap insertion / deletion (\(\delta\)), score = -1

Note

This is a linear gap score

6.13.1.2. Scored match / mismatch¶

We construct a matrix \(F\), such that \(F[i, j]\) is the score for the best alignment up to \(i,j\). \(F\) is built recursively using the following cases:

\[\begin{split}F[i,j] = \max \begin{cases} F[i-1, j-1] + score(S_1[i], S_2[j])\\ F[i-1, j] + \delta\\ F[i, j-1] + \delta\\ \end{cases}\end{split}\]

where \(score(S_1[i], S_2[j])\) is the score for the match of position \(i\) and \(j\) from sequences \(S_1\) and \(S_2\) respectively.

If we set \(i=1,j=1\), then the possible alignment paths leading to cell \(X[i,j]\) are:

  • \((i,j-1)\), from \(\leftarrow\)

  • \((i-1,j)\), from \(\uparrow\)

  • \((i-1,j-1)\), from \(\nwarrow\)

We arrive at \(F[1, 1]\), \(X\), by adding \(\delta\) to the \(F\) from each of the possible flanking cells and take the maximum of these. (\(\delta\) is the mismatch and gap insertion penalties equal \(-1\).)

    if a == b:
        val = 1
    else:
        val = -1
    return val
